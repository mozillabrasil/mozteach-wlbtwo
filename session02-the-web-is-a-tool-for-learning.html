Alfabetização Web Básica II | 2. A Web é a ferramenta de aprendizado.

Feito por Mozilla

2 Horas

Aprendizes usarão instruções web nativas para criar um meme ou construir um projeto pessoal na vida real aprendendo navegação, remixagem, e compartilhamento.

Atividade 2 de 9

Ler, Escrever e Participar da web

Competências de Alfabetização Web

Avaliação Navegação Práticas abertas Remixagem

Habilidades do Século 21

Objetivos de aprendizado

  • Utilizar instruções Web nativas para construir algo na vida real.
  • Remixar e customizar um projeto pessoal da web.
  • Identificar várias comunidades on-line de criações de práticas.

  • 1. Preparação

    Faça a atividade por conta própria para se familiarizar com ela. Clique nos links na atividade e tenter fazer um ou dois dos projetos sugeridos usando as instruções que você encontrar online.

    1. Siga este link para o Thimble a ferramenta usada nesta atividade.
    2. Clique no botão verde "Remix" no canto superior direito da janela para entrar no código do projeto.
    3. Clique no aba "Tutorial" ao lado da aba "visualizar" no canto superior direito da janela de codificação.
    4. Siga os passos do tutorial para completar a atividade. Você também pode precisar voltar aqui e completar algumas das etapas deste plano de aula para concluir com êxito o projeto.

    Você também pode querer criar uma conta (ou várias) em Thimble para que você possa compartilhar com seus alunos. Isto irá poupar tempo mais tarde, quando eles estiverem prontos para usar o novo recurso "selfie" do thimble para tirar fotos de seus projetos.

    Revise esta definição da palavra "Remix" de modo que você possa ajudar seus alunos (e colegas!) a entender que eles podem remixar seus projetos para customizar e melhorá-los.

    Poste a URL ou endereço da Web, do projeto de hoje em algum lugar bem visível no seu quarto. Você pode querer postá-lo como um link encurtado usando um serviço como bit.ly.

  • 2. Introdução

    20 minutos

    Dê as boas vindas aos seus alunos e explique que hoje estaremos usando a Web como uma ferramenta para a aprendizagem. Vamos usar o que aprendemos para construir algo incrível.

    • Escolha um projeto que queremos fazer, a partir de uma lista de escolhas para iniciantes, intermediários e avançados.
    • Aprenda a ler instruções sobre a Web para fazer um projeto na vida real.
    • Aprenda a usar a ferramenta Thimble da Mozilla para documentar e compartilhar o nosso trabalho de forma segura.
    • Reflita sobre o que podemos aprender a partir de diferentes comunidades na Web.

    Para começar, diga aos seus estudantes que vamos pensar sobre onde vamos para aprender coisas novas. Talvez alguns de nós já utilizem a Web para aprender dentro ou fora da escola. Talvez alguns de nós tênham professores ou mentores ou irmãos ou tutores a quem peçam ajuda para aprender algo novo. Alguns de nós, suspiro, podem ainda continuar a usar livros para aprender!

    Explique que nós iremos mapear novos aprendizados utilizando diferentes marcadores coloridos e cartolinas. Nós faremos uma chave juntos que represente as diferentes maneiras de aprendizado, e vamos usar a tecla para ligar nossas pequenas figuras para as coisas que aprendemos ou queremos aprender.

    Estes são os passos:

    1. Peça aos estudantes para juntarem-se em grupos de 3 ou 4, ou use uma maneira randômica(ou propositalmente) de criar grupos do mesmo tamanho.
    2. Distribua partes de cartolinas e alguns marcadores para cada grupo, ou peça que os membros peguem os materiais.
    3. Peça ao grupo para criarem pequenos bonecos de palito de si próprios no meio do papel e rotule as figuras com seus nomes.

    Peça ao grupo que crie uma lista que todos possam desenhar ao lado do seu desenho. Esta lista precisa usar diferentes cores para cada um destes caminhos de aprendizado: Pergunte a alguem, Leia Online, Leia um livro, Tenha aulas e fazer eu mesmo. Sinta-se livre para usar as cores sugeridas.

    Em seguida, dê aos grupos 5 minutos para fazerem um brainstorm com todas as coisas que aprenderam ou que eles querem aprender. Peça-lhes para escrever tudo o que eles puderem pensar no cartaz do seu grupo.

    Após o brainstorm, dê aos estudantes outros 5 minutos para ligar seus bonecos de palito com qualquer conteúdo que eles aprenderam ou querem aprender usando a cor que eles acreditam que mais possa ajuda-los.

    Uma vez que os estudantes tenham mapeado eles próprios e seus caminhos de aprendizado, peça a cada grupo para refletir por um ou dois minutos, sobre qual caminho de aprendizado - ou caminhos de aprendizado - foram mais utilizados para aprender coisas novas.

    Finalmente, peça a cada grupo que compartilhem seu cartaz e tenhm um porta-voz de cada grupo para dizer que caminho de conhecimento o grupo mais usou.

    Observe quaisquer padrões que você veja e aponte como os mapas de aprendizagem dos estudantes parecem muito com a Web, que é uma rede de compartilhamento de ideias e informações. Sugira que a nossa classe também é uma rede como essa, nós somos um aprendizado Web!

    Uma vez que cada grupo teve a oportunidade de compartilhar seu cartaz, peça aos estudantes para se sentem em frente a seus computadores e acessarem a página da web para a atividade de hoje!.

  • 3. Faça o Projeto

    45 minutos

    Agora é hora de deixar os alunos soltos para fazerem projetos utilizando as instruções de um dos projetos da atividade de hoje. Peça aos alunos para procurarem nos projetos e escolher um para tentar fazer nos próximos 45 minutos.

    Ajude os alunos que se esforçarem para fazer uma escolha ou a leitura de texto on-line e garanta que os alunos saibam que podem ajudar um ao outro. Deixe os alunos colaboram em pares, se eles se ajudam, todos são bem sucedidos com o projeto. Peça aos alunos que trabalhem em qualquer lugar na sala em que fiquem confortáveis, tenham espaço suficiente e lembre os alunos das normas da comunidade sobre de compartilhamento e valorização de materiais e suprimentos.

    Verifique o tempo a cada cinco minutos ou mais e permita de 5 a 10 minutos para limpeza no final da compilação.

    Após uma limpeza, faça os estudantes voltarem para seus computadores para remixar o projeto de hoje com as fotos que eles fizeram.

    Este também é um bom momento para fazer uma pausa e dividir a atividade entre duas classes ou sessões, se você precisar.

  • 4. Documente o projeto

    20 minutos

    Convide os estudntes para clicar no botão "Remix" na atividade de hoje. Clicando no botão will take learners into the code of the webpage that holds the activity. Peça aos estudantes para entrar nas suas contas do Thimble clicando no botão "Log in" na direita do canto mais superior da tela.

    Uma vez que os estudantes estejam logados, é hora de remixar. No lado direito da tela, above da tela de visualização, os estudantes precisam ver a aba "tutorial". Peça aos estudantes para clicarem na aba para que descubram como eles podem trocar a imagem na página web com as fotos dos seus projetos.

    Peça aos estudantes para ir aos passos 3 e 4 para pegarem fotos dos seus projetos no Thimble e troquem as figuras antigas por estas novas. Leia as direções Read the directions aloud e então ajude os estudantes que tiverem dificuldades em seguir a experiência.

    1. Clique no símbolo de mais.

    2. Escolha "Upload."

    3. Escolha "Take a selfie."

    4. Siga os passos no tutorial para inserir "selfie1.png" no seu projeto.

    Separe algum tempo para os estudantes brincarem com suas funcionalidades de selfies, mas seja claro ao afirmar que hoje nós vamos documentar nosso trabalho com a câmera e be wise sobre quantas outras imagens nós mesmos nós temos online.

    Uma vez que cada estudante ou pares de estudantes tenham uma foto tirada e inserida na página web, convide a classe para uma reflexão.

  • 5. Reflexão

    15 minutos

    Facilitar a reflexão da discussão do que e como os alunuos aprendem hoje. Use questões como essa ou desenvolva suas próprias questões que tragam os alunos a considerarem como era aprender com instruções e comunidades na Web.

    Antes de você começar, lembre os alunos das normas de bondades da comunidade e encoraja eles a falarem do seu aprendizado, não sobre outras pessoas.

    • O que funcionou para você ou pareceu divertido na aprendizagem de hoje?
    • Oque não funcionou ou pareceu difícil?
    • Como foi o aprendizado da Web de similar ou diferente das outras formas de aprender?
    • Antes da aula de hoje, algum de vocês já usou qualquer um do site site que visitamos? O que achou deles? Sua opinião mudou hoje por alguma motivo?
    • Você acha que voltaria a um dos sites da comunidade que usamos hoje para aprender alguma coisa ou então compartilhar instruções para um projeto que você inventou?
    • O que faz da Web "boa" para compartilhar, ensinar e aprender?
    • O que dificulta o compartilhamento, ensino e o aprendizado na Web?
    • O que é o mais fácil e o mais difícil de trocar sua foto hoje? O que você aprendeu alterando a imagem?
    • Você mudaria seu mapa do inicio da aula, em qualquer forma, com base no que você aprendeu online hoje?
    • O que você poderia procurar na Web sobre seu projeto depois de hoje?
    • O que poderia tentar aprender na Web sobre seu projeto depois de hoje?

    Você pode pedir aos alunos para documentar ou registrar as suas respostas da avaliação. Certifique-se de ajudar cada aluno encontrar uma maneira de compartilhar o que funciona para ele, bem como para você, para que você possa avaliar seu aprendizado sobre o tópico de hoje sem uma ferramenta especial (como papel/lápis) bloqueando a expressão de um aluno.

    Se você ou seus alunos estão curiosos para saber mais, confira este Introdução ao HTML de Mozilla Developer Network Learning Area.
