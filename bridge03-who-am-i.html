Alfabetização Web Básico II | 7. Quem sou eu?

Produzido por Mozilla

90 minutos

Os alunos irão realizar uma pesquisa de imagem inversa para encontrar informações sobre um assunto on-line e, em seguida, rever uma página com seu próprio texto e imagens, aprendizagem composição, navegação, pesquisando, e compartilhando.

  • 1. Preparação

    15 minutos

    Faça a atividade em seu próprio trabalho para se familiarizar com a base colaborativa e a prática de busca aberta.

    1. Siga este link para o Thimble projeto para esta atividade.
    2. Clique no botão verde "Remix" na parte superior no canto direito da janela para entrar no código do projeto.
    3. Clique no painel "Tutorial" próximo ao painel "Visualização" no canto superior direito da janela de codificação.
    4. Siga os passos no tutorial para completar a atividade.Você também pode precisar voltar aqui para checar e completar algumas das etapas do plano desta lição para concluir com êxito o projeto Thimble.

    Você também pode configurar contas no Thimble para que você possa compartilhar com seus alunos. Isto vai economizar tempo mais tarde quando eles estiverem prontos para remixar o projeto de hoje.

    Postar a URL, ou o endereço Web, do projeto de hoje em um lugar bem visível no seu espaço. Você pode postá-lo como um link encurtado usando um serviço como bit.ly.

  • 2. Introdução

    5 minutos

    Acolha seus alunos e explique que hoje vamos resolver um mistério online usando a busca de imagem inversa. A Busca de imagem inversa usa imagens para encontrar onde essas imagens permanecem online. Nós vamos usar a busca de imagem inversa para descobrir quem é o sujeito de uma imagem mistério. Então vamos fazer o nosso próprio "Quem sou eu?" desafios para os nosso amigos tentar resolver. Para realizar essas tarefas, vamos:

    1. Usar uma imagem comum para executar uma pesquisa de imagem inversa e encontrar informações sobre o seu assunto.
    2. Adicionar o que nós aprendemos para uma página web sobre esse assunto no Thimble, editor de código online da Mozilla.
    3. Remix aquela página web com uma nova imagem para criar outro "Quem sou eu?" desafio para seus amigos resolverem.

    Nós vamos trabalhar principalmente no Thimble hoje e usar um tutorial dentro do "Quem sou eu?" projeto para nos guiar através de todas as etapas. Os alunos podem ir para o projeto aqui.

  • 3. Condução de uma busca de Imagem Inversa

    15 minutos

    Assim que os alunos chegarem ao projeto de hoje, eles devem clicar no botão verde "Remix" no canto superior direito para acessar seu código e o tutorial. Os alunos podem clicar na área cinza "Tutorial" acima da janela de pré-visualização no lado direito da tela para encontrar instruções passo-a-passo para a atividade de hoje.

    Você pode querer acompanhar as etapas em seu computador projetando e fazendo um pensamento alto com a classe para ajudar os estudantes orientando a si mesmos para a interface do Thimble e o tutorial. As instruções para a busca de imagem inversa diz:

    1. Clique no botão cinza "Publicar" no canto superior direito desta página. Dê uma breve descrição do tipo, "Meu Quem sou eu? página do projeto," e quando confirmar a publicação isto lhe dará um link que você pode clicar para abrir o arquivo chamado index.html em uma nova aba do navegador.
    2. Clique com o botão direito do mouse ou clique sobre a imagem na página web que é aberta depois da publicação. Escolha, "Copiar o endereço da imagem," ou, "Copiar a URL da imagem."
    3. Ir para um motor de busca de imagem como Google Imagens.
    4. Dentro da caixa de pesquisa, clique no ícone câmera. Isto irá iniciar a busca de imagem inversa.
    5. Copie o endereço da imagem, ou URL, copie na caixa de pesquisa e pressione enter.
    6. Confira os resultados da pesquisa. Cada um deve apontar para você uma possível fonte da sua imagem. Clique através de poucos links que aparece adequado e seguro para sua comunidade, clube, ou sala de aula. O que você aprendeu sobre a imagem, o seu assunto, e seu autor?

  • 4. Adicionando Informações para uma página web

    30 minutos

    A seguir, ajude os estudantes seguindo alguns passos do tutorial para adicionar informações sobre o assunto da imagem para o "Quem sou eu?" na página web do Thimble. Mais uma vez, se parece útil, executar através de alguns passos para a classe usando seu computador, projetando para o modelo e editando o arquivo index.html.

    1. Clique no arquivo chamado index.html na barra lateral esquerda chamada "ARQUIVOS."
    2. Vá para linha 45 e substitua o . . . . com sua resposta para a questão, "Quem sou eu?"
    3. Vá para linha 47 e substitua o . . . . com a informação sobre o assunto da imagem.
    4. Vá para linha 49.Existe uma tag aqui é uma tag link. Cada tag leva um atributo href que diz o link do endereço, ou URL, da página web que deve ir quando clicado. Altere href="" copiando e colando o endereço,ou URL, da página web que você lê entre aspas, igual a: href="[meu endereço da página web]".Você não necessita dos parênteses, entretanto, apenas o endereço web.

  • 5. Criando seu próprio "Quem sou eu?" Desafio

    30 minutos

    Depois dos alunos publicarem seu projeto "Quem sou eu?" concluído, convida eles para criar seus próprios desafios remixados para seus amigos usando seus heróis ou personagens favoritos que podem não ser conhecidos por todos na sala. Modelar as etapas do tutorial para os alunos que necessitam de ajuda para criar nova página no Thimble.

    1. Vá em publicar novamente para salvar sua página web Quem sou eu? concluída.
    2. Clique no link Thimble para você dar como finalizado seu projeto.
    3. Uma vez que você chegar em seu projeto concluído, clique no botão verde "Remix" no canto superior direito. Isto irá levar você de volta para o código de uma nova versão da página.
    4. Altere o nome deste projeto clicando no seu nome no canto superior esquerdo da tela de codificação. O Título é no pequeno ícone do Thimble naquele canto. Mude o nome para algo como, "Meu novo Quem sou eu? Desafio."
    5. Abra um motor de busca de imagem novamente.
    6. Encontre a imagem de um de seus heróis ou personagem favorito, mas faça um herói ou personagem que outras pessoas possam não saber.
    7. Depois que você encontrar uma imagem para usar, clique através de todas as miniaturas e obtenha a imagem em sua página web original.
    8. Botão direito do mouse ou comanda o clique na imagem e copia o endereço irs ou a URL.
    9. Volte para index.html.
    10. Vá para linha 37.
    11. Encontre o img,ou a tag image. Esta tag src, ou source, atributo para chamar a página onde está a imagem. Exclua o nome da imagem original who-am-i-1.jpg.
    12. Cole o endereço de sua nova imagem.
    13. Apagar suas respostas do "Quem sou eu?" caixa ao lado da imagem.
    14. Publicar seu projeto e dar a ele uma descrição.
    15. Compartilhe o link que você começou com um amigo que está pronto para assumir seu novo Quem sou eu? desafio.

  • 6. Reflexão

    10 minutos

    Depois que os estudantes terminarem de remixar seus próprios desafios, facilite uma reflexão e discussão sobre o trabalho do dia.

    Antes de você começar, lembre os estudantes das normas das comunidades sobre a bondade e incentivar eles para conversar sobre a aprendizagem deles, não sobre outras pessoas.

    Use questões como estas, ou desenvolva suas próprias.

    • O que você aprendeu sobre busca hoje? Quando e como pode ser útil a busca de imagem inversa?
    • Você sacaneou seu amigo? O que você curtiu sobre criar seu próprio desafio? O que parece difícil sobre remixar sua própria página web no Thimble?
    • Além de usar palavras e imagens, de que outra forma você gostaria de pesquisar on-line?
    • Quais outras partes da página web como esta que você quer aprender como codificar a seguir?

    Você pode pedir aos alunos para documentar ou gravar as suas respostas para avaliação. Certifique-se de ajudar cada aluno a encontrar uma maneira de compartilhar o que funcionou para ele, bem como para você, para que você possa avaliar o seu aprendizado sobre o tópico de hoje sem uma ferramenta especial (como papel/lápis) bloqueando a expressão do aluno.

    Se você ou seus alunos são curiosos para aprender mais, confira isto Introdução para HTML da Mozilla Developer Network Área do Aluno.
